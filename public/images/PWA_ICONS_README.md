# PWA Icons Required

The following icons need to be created for the CMS PWA:

## Required Icons

### App Icons
- `cms-icon-192.png` - 192x192px - Main app icon (small)
- `cms-icon-512.png` - 512x512px - Main app icon (large)
- `cms-badge.png` - 96x96px - Badge icon for notifications

### Shortcut Icons
- `shortcut-dashboard.png` - 96x96px - Dashboard shortcut
- `shortcut-invoice.png` - 96x96px - New invoice shortcut
- `shortcut-customers.png` - 96x96px - Customers shortcut

### Screenshots (Optional but Recommended)
- `cms-screenshot-1.png` - 1280x720px - Dashboard overview
- `cms-screenshot-2.png` - 1280x720px - Invoice management

## Design Guidelines

### Colors
- Primary: #2563eb (blue-600)
- Background: #ffffff (white)
- Text: #111827 (gray-900)

### Style
- Modern, clean design
- Use the building/office icon theme
- Rounded corners (16px radius for 192px icon)
- Consistent with CMS branding

### Tools
- Figma, Adobe Illustrator, or Sketch
- Export as PNG with transparency
- Optimize with TinyPNG or similar

## Temporary Workaround

Until icons are created, the PWA will still work but:
- Install prompt will show browser default icon
- Home screen icon will be generic
- Shortcuts won't have custom icons

The functionality is not affected, only the visual appearance.

## Quick Generation

You can use online tools to generate icons:
- [PWA Builder](https://www.pwabuilder.com/)
- [RealFaviconGenerator](https://realfavicongenerator.net/)
- [App Icon Generator](https://appicon.co/)

Upload a 512x512px master icon and these tools will generate all sizes.
