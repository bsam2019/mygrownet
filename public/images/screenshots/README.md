# PWA Screenshots

## Required Screenshots

Add 3 screenshots here for the PWA install prompt:

1. **dashboard.png** - Dashboard view (1080x1920 portrait)
2. **wallet.png** - Wallet screen (1080x1920 portrait)
3. **network.png** - Network tree (1080x1920 portrait)

## How to Create Screenshots

### Method 1: Take Real Screenshots
1. Open MyGrowNet on your phone or Chrome DevTools mobile view
2. Navigate to Dashboard, Wallet, and Network pages
3. Take screenshots
4. Resize to exactly 1080x1920 pixels
5. Save as PNG files with the names above

### Method 2: Use Chrome DevTools
1. Open Chrome DevTools (F12)
2. Click device toolbar (Ctrl+Shift+M)
3. Set device to "Pixel 5" or custom 1080x1920
4. Navigate to each page
5. Right-click → "Capture screenshot"
6. Save with correct names

### Method 3: Online Screenshot Tool
Use a tool like:
- https://www.screely.com/
- https://screenshots.cloud/
- https://mockuphone.com/

## Requirements
- **Size**: 1080x1920 pixels (portrait)
- **Format**: PNG
- **Quality**: High resolution, clear UI
- **Content**: Show actual app interface, not logos or placeholders

## Why This Matters
Chrome Android uses these screenshots in the install prompt. Good screenshots significantly improve install rates.
