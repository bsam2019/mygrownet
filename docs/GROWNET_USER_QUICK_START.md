# GrowNet User Quick Start Guide

**Last Updated:** March 4, 2026
**Status:** Production
**For:** New and existing GrowNet members

## Overview

This is your practical guide to using the GrowNet platform effectively. Whether you're brand new or need a refresher, this guide covers the essential features you'll use daily.

## Getting Started (First 24 Hours)

### 1. Access Your Dashboard

**Desktop:** Visit mygrownet.com and log in
**Mobile:** Use the mobile-optimized view (toggle in settings)

Your dashboard shows:
- Total earnings at a glance
- Team size and growth
- Current professional level
- Quick actions

### 2. Complete Your Profile

**Why it matters:** Your phone number is required for mobile money withdrawals.

**Steps:**
1. Click your name (top right) → Profile
2. Add your phone number (format: +260 9XX XXX XXX)
3. Verify your email if not done
4. Add a profile photo (optional but recommended)

### 3. Get Your Referral Link

**Steps:**
1. Dashboard → "My Network" section
2. Copy your unique referral link
3. Share with friends via WhatsApp, SMS, or social media

**Your link looks like:** `mygrownet.com/register?ref=YOUR_CODE`

## Understanding Your Dashboard

### Key Metrics (Top Cards)

**Total Earnings**
- All-time earnings from commissions, bonuses, and profit-sharing
- Click to see detailed breakdown

**Team Size**
- Total people in your 7-level network
- Includes all active and inactive members

**Team Volume**
- Combined subscription value of your network
- Resets monthly

**Active Members**
- Team members who qualified this month
- Must meet minimum activity points

### Professional Level Progress

Shows your current level (Associate → Ambassador) and progress to next level.

**What you need:**
- Lifetime Points (LP) - Never expire
- Monthly Activity Points (MAP) - Reset monthly
- Team requirements (varies by level)

## Essential Features Guide

### 💰 Wallet & Earnings

**View Balance:**
Dashboard → Wallet card (shows available balance)

**Withdraw Money:**
1. Wallet → "Withdraw"
2. Enter amount (minimum K50)
3. Select Mobile Money (MTN or Airtel)
4. Confirm phone number
5. Submit request
6. Wait for admin approval (usually 24-48 hours)

**Important:**
- Withdrawals go to your registered phone number
- Ensure phone number is correct in profile
- Check transaction history for status

### 👥 Building Your Network

**Invite Members:**
1. Get your referral link (Dashboard → Share button)
2. Share via WhatsApp, SMS, or social media
3. Help them complete registration
4. They automatically join your network

**View Your Team:**
1. Dashboard → "Team" tab
2. See all levels (1-7)
3. Filter by active/inactive
4. View individual member details

**Matrix Position:**
- You can have 3 direct referrals (Level 1)

- Everyone benefits from spillover

## Monthly Qualification

### Why It Matters

To receive commissions and bonuses each month, you must qualify by earning minimum Monthly Activity Points (MAP).

### How to Qualify

**Earn MAP through:**
- Personal subscription (automatic points)
- Referring new members
- Team activity
- Platform engagement
- Attending workshops

**Minimum MAP by Level:**
- Associate: 100 MAP
- Professional: 200 MAP
- Senior: 300 MAP
- Manager: 400 MAP
- Director: 500 MAP
- Executive: 600 MAP
- Ambassador: 800 MAP

**Check Your Status:**
Dashboard → Points card shows current MAP and requirement

## Troubleshooting

### Can't Log In

**Solutions:**
1. Check email/phone number is correct
2. Try "Forgot Password" link
3. Clear browser cache
4. Try different browser
5. Contact support if still stuck

### Withdrawal Not Received

**Check:**
1. Wallet → Transaction History
2. Look for status: Pending/Approved/Completed
3. If "Pending" - wait for admin approval
4. If "Approved" - check phone number is correct
5. If "Completed" - check mobile money account
6. Contact support with transaction ID

### Referral Not Showing

**Verify:**
1. New member used your referral link
2. They completed registration
3. They activated subscription
4. Wait 5-10 minutes for system update
5. Refresh dashboard
6. Check "Team" tab → Level 1

### Points Not Updating

**Solutions:**
1. Refresh dashboard (F5 or pull down on mobile)
2. Check "Points History" for details
3. Verify qualifying activities
4. Wait for monthly calculation (1st of month)
5. Contact support if discrepancy

### Dashboard Not Loading

**Try:**
1. Clear browser cache and cookies
2. Try incognito/private mode
3. Update browser to latest version
4. Try different device
5. Check internet connection
6. Contact support if persists
