### Posting Schedule
- **Monday**: Feature spotlight or tip
- **Wednesday**: App-specific card (rotate through apps)
- **Friday**: Success story or testimonial
- **Weekend**: CTA or value proposition card

### Hashtag Strategy
```
#MyGrowNet #ZambiaBusiness #BusinessGrowth #SmallBusinessZambia 
#Entrepreneurship #BusinessTools #StartupZambia #GrowYourBusiness
#BusinessManagement #DigitalBusiness
```

### Platform-Specific Adaptations
- **Facebook**: Use 1200x630px cards, longer captions
- **Instagram**: Use 1080x1080px cards, concise captions, stories
- **LinkedIn**: Professional tone, focus on business value
- **Twitter/X**: Create 1200x675px versions, thread format

## Design Specifications

### Color Palette
- **Primary Blue**: #2563eb
- **Success Green**: #059669
- **BizBoost Purple**: #7c3aed
- **Marketplace Orange**: #ea580c
- **GrowBuilder Indigo**: #4f46e5

### Typography
- **Headlines**: Bold, sans-serif (Inter, Poppins, or similar)
- **Body**: Regular weight, high readability
- **Minimum font size**: 24px for mobile viewing

### Logo Usage
- Always include MyGrowNet logo
- Maintain clear space around logo
- Use white logo on dark backgrounds
- Use colored logo on light backgrounds
