## What's the Module System?

The module system allows MyGrowNet to offer:
1. **Core MLM Dashboard** - FREE for all members
2. **Standalone Apps** - Subscription-based tools (Wedding Planner, Business Plan Generator, etc.)
3. **Business Tools** - SME accounting, inventory, CRM
4. **Multiple User Types** - MEMBER, CLIENT, BUSINESS, INVESTOR, EMPLOYEE
