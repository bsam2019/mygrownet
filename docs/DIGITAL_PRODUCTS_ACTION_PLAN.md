## üìù Content Creation Guidelines

### E-Books (40-60 pages each)

**Structure:**
1. Cover page with MyGrowNet branding
2. Table of contents
3. Introduction (2-3 pages)
4. Main content (30-50 pages)
   - Break into 5-7 chapters
   - Use headings and subheadings
   - Include bullet points and lists
   - Add actionable tips
5. Conclusion (2-3 pages)
6. Resources/References (1-2 pages)

**Design Tips:**
- Use consistent fonts (Arial, Calibri, or similar)
- Add MyGrowNet logo on each page header/footer
- Use colors from brand palette (blues, greens)
- Include images/diagrams where helpful
- Add page numbers

**Tools:**
- Microsoft Word
- Google Docs
- Canva (for design)
- Adobe InDesign (if you have it)

### Videos (5-20 minutes each)

**Structure:**
1. Intro (30 seconds)
   - MyGrowNet logo
   - Video title
   - Your name/role
2. Main content (4-18 minutes)
   - Clear sections
   - Visual aids (slides, screen shares)
   - Examples and demonstrations
3. Outro (30 seconds)
   - Key takeaways
   - Call to action
   - MyGrowNet branding

**Recording Tips:**
- Use good lighting
- Clear audio (use microphone if possible)
- Minimize background noise
- Speak clearly and at moderate pace
- Use slides or screen recording
- Edit out mistakes

**Tools:**
- Zoom (record meetings)
- Loom (screen recording)
- OBS Studio (free, professional)
- Camtasia (paid, easy to use)
- DaVinci Resolve (free editing)

### Templates

**Social Media Pack:**

**Presentation Deck:**
